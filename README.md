# You-Are-a-Cosmic-Dream-Twine
Disclaimer: This is the html file for my in-development Twine game. It is a work in process so the file fill be regularly updated.

*You Are a Cosmic Dream* is a Twine (Harlowe 3.3.8) interactive fiction game exploring posthuman memory, grief, and relationality. Players inhabit one of three more-than-human figures navigating symbolic landscapes and branching narrative paths.

## How to Play
- Download `You Are a Cosmic Dream.html` into your device.
- Open the html file in a web browser.

- Anglerfish path currently in development; other paths awaiting action.
